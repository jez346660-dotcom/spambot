# TikTok Comment Bot (Telegram)

⚠️ **ОБРАЗОВАТЕЛЬНЫЙ ПРОЕКТ**

Telegram-бот для автоматического комментирования видео в TikTok.
Работает через браузерную автоматизацию с Playwright.

## ⚠️ ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ

Массовая автоматическая рассылка комментариев нарушает правила TikTok.
Этот проект создан **исключительно в образовательных целях**.
Используйте только с согласия владельцев аккаунтов и видео.

## 🚀 Возможности

- Вход в TikTok через cookies или логин/пароль
- Автоматическое комментирование 5-35 видео
- Случайный выбор текста комментария из 2-3 вариантов
- Имитация поведения человека (задержки, скролл, опечатки)
- Антидетект меры (user-agent, viewport, скрытие webdriver)
- Работа 24/7 на Render (бесплатный тир)

## 📦 Установка

### Локальный запуск

```bash
# Клонируем репозиторий
git clone https://github.com/your-username/tiktok-spam-bot.git
cd tiktok-spam-bot

# Устанавливаем зависимости
pip install -r requirements.txt

# Устанавливаем браузер для Playwright
playwright install chromium
playwright install-deps

# Создаём .env файл с токеном бота
echo "BOT_TOKEN=ваш_токен_бота" > .env

# Запускаем
python main.py
